## üé® What Are ASE Files?

**ASE (Adobe Swatch Exchange)** is a universal color swatch format developed by Adobe that allows you to share color palettes across multiple Adobe applications including:

- **Adobe Photoshop**
- **Adobe Illustrator**
- **Adobe InDesign**
- **Adobe Fresco**
- **Adobe XD**
- And other Adobe Creative Cloud applications

ASE files store colors with their exact RGB/CMYK values and names, ensuring perfect color consistency across projects and applications.

---

## üí° Why This Collection Exists

### The Problem

Digital graffiti artists and illustrators often want to create artwork that accurately represents what they could paint with real spray cans. However, finding and manually entering hundreds of exact spray paint colors is:

- **Time-consuming** - Manually adding each color takes hours
- **Error-prone** - Easy to mistype hex codes or RGB values
- **Tedious** - No easy way to import entire manufacturer color ranges

### The Solution

This repository provides **ready-to-use ASE files** containing the complete color ranges from major spray paint manufacturers. Simply download and import them into your Adobe apps to instantly have access to authentic spray paint colors.

### Use Cases

- **Digital Graffiti Art** - Paint digitally with colors that match your physical cans
- **Concept Art** - Plan murals and pieces before buying paint
- **Color Matching** - Find the perfect spray paint color for your digital artwork
- **Design Mockups** - Create realistic spray paint effect designs
- **Education** - Learn color theory with real-world spray paint palettes

## üì• Importing into Adobe Applications

### Adobe Photoshop

1. Open Photoshop
2. Go to **Window ‚Üí Swatches** to open the Swatches panel
3. Click the **panel menu icon** (‚ò∞) in the top-right corner
4. Select **Import Swatches...**
5. Navigate to your downloaded `.ase` file
6. Click **Open**
7. The swatches will appear in your Swatches panel

**Pro Tip**: You can create a dedicated folder for spray paint swatches to keep them organized.

### Adobe Fresco

1. Open Fresco
2. Open the **Libraries** panel
3. Click the **menu icon** (‚ãØ)
4. Select **Import Swatches**
5. Choose your downloaded `.ase` file
6. Click **Open**
7. The colors will appear in your color library

### Adobe Illustrator

1. Open Illustrator
2. Go to **Window ‚Üí Swatches** to open the Swatches panel
3. Click the **Swatch Libraries menu** icon
4. Select **Other Library...**
5. Navigate to your `.ase` file
6. Click **Open**
7. A new panel with the swatches will appear

### Adobe InDesign

1. Open InDesign
2. Go to **Window ‚Üí Color ‚Üí Swatches**
3. Click the **panel menu icon** (‚ò∞)
4. Select **Load Swatches...**
5. Choose your `.ase` file
6. Click **Open**

### ASE File Format

ASE files use a binary format:

```
Offset  Type      Description
------  --------  -----------
0x00    char[4]   "ASEF" signature
0x04    uint16    Version major (1)
0x06    uint16    Version minor (0)
0x08    uint32    Number of color blocks
...     varies    Color blocks (name + RGB + type)
```

Each color block contains:
- Color name (UTF-16BE encoded)
- Color model ("RGB ")
- RGB values (float32, 0.0-1.0)
- Color type (Normal/Spot/Global)
